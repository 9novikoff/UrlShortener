About URL Shortener

Welcome to the URL Shortener app! This application allows you to shorten long URLs and manage them easily. You can create, view, and delete shortened URLs effortlessly.

Get started by shortening your first URL!

Our URL shortener generates unique, compact links by hashing the original URL with SHA-256 and converting the result to a URL-safe Base64 string. The shortened link is created by taking the first 8 characters of this string, ensuring it is both secure and easy to use.
